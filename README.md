## Base image for opencv and dlib tasks
Buildpack deps based on a stable Debian:Jessie distribution, additionally there is a latest OpenCV installed from apt and a 19.4 dlib library built from source.

#### Contains:
- OpenCV == latest
- DLib == 19.4
